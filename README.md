## Описание

Strictly - виджеты для отображения курсов валют.

Стэк: SwiftUI, WidgetKit.

API бесплатное на 7 дней, доступно на сайте https://console.fastforex.io. Чтобы отображались данные, нужно поменять apiKey на новый в классе NetworkService.
